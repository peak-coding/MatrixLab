# MatrixLab

MatrixLab is a small Python project I built to strengthen my understanding of linear algebra and how it connects to artificial intelligence.

As a CSE (Data Science & AI) student, I wanted something practical that goes beyond solving textbook problems. This toolkit implements core matrix operations and visualizes basic linear transformations to better understand the math used in machine learning.

---

## What This Project Currently Does

- Matrix addition and multiplication  
- Determinant calculation  
- Eigenvalues and eigenvectors (using NumPy)  
- Basic 2D linear transformation visualization  
- Simple vector operations  

This is an evolving project, and I plan to expand it as I progress in my coursework.

---

## Why I Built This

Linear algebra is fundamental to AI:

- Neural networks rely heavily on matrix multiplication.
- PCA depends on eigenvectors and eigenvalues.
- Optimization and gradient methods operate on vectors.

Instead of just studying theory, I wanted to implement these concepts directly in Python to understand what’s happening under the hood.
